# 易自习作业管理系统

## Project setup
```
npm install
```
> or(if you use yarn as package manager)
```bash
yarn
```


### Compiles and hot-reloads for development
```
npm run serve
```
> or(if you use yarn as package manager)
```bash
yarn serve
```
### Compiles and minifies for production
```
npm run build
```
> or(if you use yarn as package manager)
```bash
yarn build
```
### Lints and fixes files
```
npm run lint
```
> or(if you use yarn as package manager)
```bash
yarn lint
```
